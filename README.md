## 贪吃蛇小游戏

已新增一个纯前端的贪吃蛇小游戏。

- 文件：`index.html`
- 在线运行：可通过 GitHub Pages 访问

## 操作说明

- 点击“开始 / 重新开始”开始游戏
- 使用方向键或 `W/A/S/D` 控制蛇的移动
- 点击“暂停”可暂停或继续
- 撞墙或撞到自己会结束游戏

## 难度与排行榜

游戏分为三级：

- 低级：速度最慢
- 中等：速度中等
- 高级：速度最快

每个难度有独立排行榜（前 5 名），互不影响。  
排行榜数据会保存在浏览器本地存储中。
